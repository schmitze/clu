# Hypotheses

### HYP-001 – Semi-automatic memory is the sweet spot
- **Date:** 2026-03-13
- **Hypothesis:** Agent-proposed, human-confirmed memory writes will produce higher quality memory than either fully manual or fully automatic approaches.
- **Evidence for:** Manual is too much friction; automatic produces noise (see FND-001)
- **Evidence against:** Confirmation fatigue could lead to rubber-stamping
- **Status:** testing
- **How to test:** Use the system for 2 weeks, compare memory quality with a fully manual control period
